试用java流集合 -> map 时，一定要预防重复Key的情况.

不合规代码示例

class MyClass {
  static void foo1(){
    List stream = Stream.of(1, 2, 3, 4).toList();
    Map mapValue = stream.stream().collect(Collectors.toMap(key -> key, Function.identity()));
  }
}

合规解决方案

class MyClass {
  static void foo2(){
    List stream = Stream.of(1, 2, 3, 4).toList();
    Map mapValue2 = stream.stream().collect(Collectors.toMap(key -> key, Function.identity(), (key1, ke2) -> key1));
  }
}
